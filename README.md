# MOS SeaBIOS

MOS SeaBIOS provides **built and packaged SeaBIOS firmware** for use within the MOS ecosystem.

This repository contains the **build scripts, packaging logic, and configuration files**
required to compile SeaBIOS from upstream sources and package it for MOS.

---

## Overview

SeaBIOS is built **from upstream source code** and packaged for integration into MOS.

No functional changes to SeaBIOS itself are intended.  
The goal of this repository is to provide **consistent, reproducible SeaBIOS packages** for MOS installations.
